# Exp-1 ATM & BANKING SYSTEM

# REG NO : 212224230039
# NAME : BLESSING S

# AIM:
To design and develop an ATM and Banking System that automates essential banking operations such as cash withdrawal, deposit, balance inquiry, fund transfer, and transaction management, ensuring secure and efficient financial transactions for customers.

# SRS (Procedure):
Identify System Objectives and User Requirements:

Determine the functional requirements for ATM users (customers) and bank administrators.

Include key features like authentication, cash withdrawal, fund transfer, and account information access.

Design the Database:

Create tables for customer details, account information, transactions, ATM machine data, and PIN verification.

Ensure data normalization and secure storage of sensitive data.

Develop User Interfaces:

Design separate interfaces for customers (ATM screen) and administrators (bank control panel).

Provide intuitive and user-friendly menus for all operations.

Implement Core Functional Modules:

Authentication Module: Verify card number and PIN.

Transaction Module: Handle withdrawal, deposit, balance inquiry, and fund transfer.

Record Module: Store and update transaction logs.

Integrate Security Features:

Implement encryption for PINs and account details.

Add measures to prevent unauthorized access and fraud.

Test and Validate the System:

Conduct testing for transaction accuracy, fault handling, and system reliability.

Check for issues such as incorrect balance updates or transaction failures.

Deploy and Maintain:

Install the system on ATM terminals and link to the central bank database.

Perform regular maintenance and software updates for better performance and security.
